0x03 踩坑记录

语言显示异常

NexT 的语言文件命名是 zh-CN.yml，如果写 zh-Hans 会匹配不到，回退到字母序第一个语言（孟加拉语 bn），导致界面文字乱码。

解决：_config.yml 中 language: zh-CN

GitHub 邮箱隐私保护

如果 GitHub 开启了 “Block command line pushes that expose my email”，推送时使用私人邮箱会被拒绝。

解决：使用 GitHub 提供的 noreply 邮箱：

Token 不要写在配置文件里

hexo-deployer-git 支持在 repo URL 中嵌入 token，但这会泄露。建议用 SSH key 或 git credential helper。
