0x00 类型系统

1. any、unknown、never 的区别？

  • any：关闭类型检查，会污染后续推导。适合快速原型或渐进迁移
  • unknown：安全的”任意类型”。必须先类型收窄才能操作
  • never：表示永远不会发生的类型（抛出异常、死循环、穷尽性检查）

2. type 和 interface 怎么选？

维度 type interface
扩展 & 交叉 extends
声明合并 ❌ ✅（同名自动合并）
联合类型 ✅ type A = B | C ❌
元组 ✅ type T = [string, number] ❌（可用但丑）
映射类型 ✅ ❌

选型原则：描述对象形状用 interface，需要联合/交叉/映射用 type。
